# Altersvorsorge-Förderrechner

Ein interaktiver Rechner für die **private Altersvorsorge** in Deutschland, inspiriert von der **aktuellen Reform der staatlichen Förderung**.  
Er zeigt, wie viel Eigenanteil, Grundförderung und Kinderförderung im Laufe der Jahre angespart werden können.

---

## Funktionen

- Eingabe von:
  - Aktuelles Alter
  - Anzahl der Kinder
  - Monatliche Sparrate
  - Startjahr der Einzahlung
- Dynamische Erstellung von Eingabefeldern für das Alter der Kinder
- Berechnung:
  - Grundförderung (je nach Startjahr)
  - Kinderförderung (nur Kinder unter 18 Jahre)
  - Kapital mit und ohne Rendite (7% p.a.)
- Fehlerbehandlung: Ungültige Eingaben werden sofort angezeigt
- Responsive und benutzerfreundliche Oberfläche
